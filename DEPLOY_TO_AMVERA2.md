# Инструкция по деплою на Amvera2 (аккаунт grut)

## Текущие настройки

- **Remote:** `amvera2`
- **URL:** `https://grut@git.msk0.amvera.ru/grut/vgltu`
- **Аккаунт:** grut

## Шаги для деплоя

### 1. Добавить все изменения в git

```bash
git add .
```

### 2. Закоммитить изменения

```bash
git commit -m "Улучшения производительности, админ-панели и тесты"
```

### 3. Отправить на amvera2

```bash
git push amvera2 master
```

Или если ветка называется по-другому:

```bash
git push amvera2 master:main
```

### 4. Проверить в панели Amvera

1. Зайти на https://amvera.ru
2. Войти в аккаунт `grut`
3. Найти проект `vgltu`
4. Проверить что сборка запустилась
5. Дождаться завершения сборки
6. Проверить логи запуска

## Переменные окружения в Amvera

Убедитесь, что в панели Amvera установлены следующие переменные:

- `BOT_TOKEN` - токен бота от BotFather
- `ADMIN_ID` - ваш Telegram ID (опционально)
- `BASE_URL_SCHEDULE` - URL API расписания (по умолчанию: https://kis.vgltu.ru/schedule)
- `BASE_URL_LIST` - URL API списков (по умолчанию: https://kis.vgltu.ru/list)

## Проверка после деплоя

1. Проверить логи в панели Amvera
2. Отправить команду `/start` боту
3. Проверить что бот отвечает
4. Проверить админ-панель командой `/admin`

## Если возникли проблемы

1. Проверить логи сборки в Amvera
2. Проверить логи запуска
3. Убедиться что все переменные окружения установлены
4. Проверить что файл `amvera.yaml` присутствует в репозитории
