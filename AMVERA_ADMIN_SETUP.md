# Настройка администратора на Amvera

## Проблема
У вас нет доступа к админ-панели, потому что ваш Telegram ID не добавлен в базу данных администраторов.

## Решение: Добавить админа через переменную окружения

### Шаг 1: Узнайте свой Telegram ID

Ваш Telegram ID: **1003795435** (из предыдущих сообщений)

Если нужно проверить:
- Используйте бота [@userinfobot](https://t.me/userinfobot)
- Или посмотрите в логах бота при отправке `/start`

### Шаг 2: Добавьте переменную окружения в Amvera

1. Войдите в панель управления Amvera
2. Перейдите в ваш проект → **Настройки** → **Переменные окружения**
3. Добавьте новую переменную:
   - **Ключ**: `ADMIN_ID`
   - **Значение**: `1003795435` (ваш Telegram ID)
4. Сохраните изменения

### Шаг 3: Перезапустите приложение

После добавления переменной:
1. Перейдите в раздел **Управление** (Management)
2. Нажмите **Перезапустить** (Restart)

### Шаг 4: Проверьте логи

После перезапуска в логах должно появиться:
```
✅ Администратор 1003795435 автоматически добавлен из переменной окружения ADMIN_ID
```

### Шаг 5: Проверьте доступ

Отправьте команду `/admin` в боте - должна открыться админ-панель!

## Альтернативный способ (если есть SSH доступ)

Если у вас есть SSH доступ к серверу Amvera:

```bash
cd /app
python add_admin.py 1003795435
```

## Что дальше?

После добавления админа вы сможете:
- ✅ Использовать команду `/admin` для доступа к админ-панели
- ✅ Управлять статусом бота (включить/выключить)
- ✅ Просматривать статистику пользователей
- ✅ Делать массовые рассылки
- ✅ Управлять другими администраторами

## Важно

⚠️ После добавления переменной `ADMIN_ID` и перезапуска, админ будет добавлен автоматически при каждом запуске (если его еще нет в базе).

✅ Это безопасно - если админ уже существует, он не будет добавлен повторно.
