# Telegram Bot для расписания ВГЛТУ

Бот для получения расписания занятий студентов и преподавателей Воронежского государственного лесотехнического университета.

## 🚀 Возможности

- 📅 Получение расписания для студентов и преподавателей
- 🔍 Поиск групп и преподавателей
- 📊 Экспорт расписания в различных форматах (изображение, PDF)
- 🔔 Настройка ежедневных уведомлений
- ⚙️ Админ-панель для управления ботом
- 🌐 Поддержка высокой нагрузки (многопоточность)

## 📋 Требования

- Python 3.9+
- Telegram Bot Token

## 🔧 Установка

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd Бот-расписание-вглту
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Настройте конфигурацию:
Создайте файл `app/config.py`:
```python
TOKEN = "YOUR_BOT_TOKEN_HERE"
```

4. Запустите бота:
```bash
python new_VGLTU_bot.py
```

## 🧪 Тестирование

Запустите все тесты:
```bash
python tests/run_all_tests.py
```

Или отдельные тесты:
```bash
python tests/test_imports.py
python tests/test_syntax.py
python tests/test_utils.py
python tests/test_database.py
python tests/test_app_creation.py
```

## 🔐 Админ-панель

Для доступа к админ-панели используйте команду `/admin`. По умолчанию администратором является пользователь с ID, указанным в `app/admin/database.py`.
